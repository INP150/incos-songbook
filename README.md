## Další písničky

Pro přidání písniček stačí odělat jednu z následujících věcí (seřazeno dle preference):

- Vytvořit MR obsahující vaší píseň.

- Vytvořit ticket s jménem písně a autorem.

- Napsat jednomu ze správců repozitáře na jakékoliv platformě s vlastním požadavkem.
